用 ESP32-S3 自带的 USB OTG 模拟 CANable，

ESP32-S3 TinyUSB SLCAN CAN 2.0B PlatformIO WebUSB
📦 GitHub 仓库 ⚡ WebUSB 刷入

🔌 这是什么？

标准 CANable 用的是 STM32 + candleLight 固件，需要 Zadig 装 WinUSB 驱动。本项目用 ESP32-S3 内置 USB OTG + TinyUSB 方案，模拟 CANable 的 VID/PID，系统自动识别为 CDC ACM 串口，免驱即用。

支持完整 SLCAN 协议（socketcand / can-utils / python-can / BusMaster 等上位机兼容），ESP32-S3 跑 TWAI 控制器直接驱动 CAN 收发器。

🚀 快速上手

方式一：WebUSB 刷入（推荐）

不用装任何工具 → 打开刷写页面 → USB 插上 ESP32-S3 → 进下载模式 → 点 Install

方式二：命令行编译

git clone https://github.com/l2015/esp32s3-slcan-canable.git
cd esp32s3-slcan-canable
pio run
pio run --target upload --upload-port COM6

🔗 上位机兼容

  • candleLight — 原生兼容，系统识别为 CANable 设备
  • socketcand — SLCAN 协议直连
  • can-utils — slcan_attach + slcand
  • python-can — can.interfaces.slcan
  • BusMaster — 选 SLCAN 接口
  • SavvyCAN — 选 SLCAN 接口
