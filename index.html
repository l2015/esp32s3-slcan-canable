🔌 What Is This?

标准 CANable 用的是 STM32 + candleLight 固件，Windows 需要 Zadig 工具手动装 WinUSB 驱动。本项目用 ESP32-S3 内置 USB OTG + TinyUSB 方案，模拟 CANable 的 VID/PID（0x16D0:0x117E），系统自动识别为 CDC ACM 串口，免驱即用。

Standard CANable uses STM32 + candleLight firmware, requiring Zadig to install WinUSB driver on Windows. This project uses ESP32-S3 built-in USB OTG + TinyUSB, emulating CANable VID/PID (0x16D0:0x117E). The system auto-detects it as a CDC ACM serial port — no driver needed.

支持完整 SLCAN 协议，兼容 socketcand / can-utils / python-can / BusMaster 等所有主流 CAN 上位机。ESP32-S3 内置 TWAI 控制器直接驱动外部 CAN 收发器。

Full SLCAN protocol support, compatible with socketcand, can-utils, python-can, BusMaster, and all major CAN tools. ESP32-S3 built-in TWAI controller directly drives external CAN transceiver.

🚀 快速上手

🚀 Quick Start

方式一：WebUSB 刷入（推荐）

不需要安装任何编译工具 → 打开刷写页面 → USB-OTG 口连接 ESP32-S3 → 进下载模式（按住 BOOT → 按 RST → 松开 BOOT）→ 点 Install

Method 1: WebUSB Flash (Recommended)

No build tools needed → Open flasher page → Connect ESP32-S3 via USB-OTG → Enter boot mode (hold BOOT → press RST → release BOOT) → Click Install

方式二：命令行编译烧录

Method 2: CLI Build & Flash

git clone https://github.com/l2015/esp32s3-slcan-canable.git
cd esp32s3-slcan-canable
pio run
pio run --target upload --upload-port COM6
