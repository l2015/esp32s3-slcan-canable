ESP32-S3 CANable SLCAN

WebUSB Firmware Flasher

WebUSB 固件刷写

  1. Connect ESP32-S3 via USB-OTG port to this computer
  2. Hold BOOT, press RST, release BOOT
  3. Click the Connect button below
  4. Select your ESP32-S3 from the browser popup
  5. Click Install to flash the firmware
  1. 通过 USB-OTG 口将 ESP32-S3 连接到电脑
  2. 按住 BOOT → 按 RST → 松开 BOOT
  3. 点击下方 Connect 按钮
  4. 在浏览器弹出的设备列表中选中 ESP32-S3
  5. 点击 Install 开始刷写固件

🔧 GPIO Wiring (SN65HVD230)

🔧 GPIO 接线 (SN65HVD230)

GPIO 4 → TXD GPIO 5 → RXD 3.3V → VCC GND → GND CANH → CAN_H CANL → CAN_L
⚠️ Use the USB-OTG port (not USB-Serial/JTAG).
First-time flashing requires manual boot mode entry.
After firmware is installed, WebUSB can update via the SLCAN CDC port.
⚠️ 请使用 USB-OTG 接口（不是 USB-Serial/JTAG 口）。
首次烧录需要手动进入下载模式。
固件刷入后，WebUSB 可通过 SLCAN CDC 端口直接更新。
Requires Chrome / Edge 89+ on desktop
需要 Chrome / Edge 89+ 桌面版浏览器
