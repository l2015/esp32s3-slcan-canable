ESP32-S3 CANable SLCAN

WebUSB Firmware Flasher

  1. Connect ESP32-S3 via USB-OTG port to this computer
  2. Hold BOOT, press RST, release BOOT
  3. Click the Connect button below
  4. Select your ESP32-S3 from the browser popup
  5. Click Install to flash the firmware
⚠️ Use the USB-OTG port (not USB-Serial/JTAG).
First-time flashing requires manual boot mode entry. After the firmware is installed, future updates can use the SLCAN serial port.
Requires Chrome / Edge 89+ on desktop
Firmware: CANable SLCAN v1.1.0 · ESP32-S3 · VID 0x16D0 PID 0x117E
